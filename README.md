# 自動顔認証プレーヤ名表示

## システム概要

格闘ゲームなどでOBSを使用して対戦の配信をする際  
対戦中のプレイヤー名を都度変更する場合に、いちいち手動での変更が大変  
本システムを導入することによって、プレイヤー名表示の簡略化することが出来ます。  
(プレイヤー名が表示されることによって、配信をご覧の視聴者も更にお楽しみいただけると思います)

## システム詳細

顔データを自動認識して数値化  
類似データがあればそちらのプレイヤー名を表示（テキスト出力→OBS連携）  
**※本システムで、顔画像が配信に出ることはありません。**

## 個人情報保護

顔データを数値化して保存しています(こちらのデータは普通に見てわからないデータとなっております)  
⇒プライバシーの観点からして顔データ数値化保存に同意して頂けない方は本システムを利用しないでください
また一時的に認証した画像を保存しておりますが、システム終了時には削除しています。

**※念のため顔情報から名前を登録される方には、（必要であれば）許可取っていただければと思います）**

## システムに必要なハード（導入するハードル）

* USB Webカメラ(1080pぐらいの解像度推奨) × 2 ＆ Windowsハード認識  
(対戦台を想定しているので2台までしか対応しておりません)
```
参考テストで使用したカメラ（リンク切れの可能性あり）

```
* ゲームセンターなどで使用する場合は、カメラ設置の許可

## システムに必要なソフト

Windows11  (たぶんWindows10でも動きます)

https://learn.microsoft.com/ja-jp/cpp/windows/latest-supported-vc-redist?view=msvc-170  
x64

## 使い方（ハード・ソフト完了後）

こちらの[FacialRecognitionSystem.zip](https://drive.google.com/file/d/1jKI5qWl4ep5TFUKK-yF3NzyXsTRV0T6f/view?usp=sharing)をダウンロード・任意のフォルダに解凍  
FacialRecognitionSystem.exe を実行

システム起動後

* ① カメラIDを適当に変更して、2台のカメラ画像が表示される事を確認  
※カメラが確認されない場合は、カメラが認識されているか確認してください  
* ② 顔認証されたプレイヤー名を登録（登録ボタンでプレイヤー名登録可）
* ③ デフォルト設定では "（インストールフォルダの）\\Data\\user1.txt" or "user2.txt" に該当するプレイヤー名が表示されるので、そちらのテキストファイルを OBSと連携(OBSの設定で適度な位置に変更)


## 使い方（高度な設定）

* プレーヤを確認する間隔(秒)　： 顔認証する間隔を変更できます。
* 参加プレーヤの一覧ファイルを作成　： 顔認証した全プレイヤーを表示(こちらを利用することによって参加メンバーが自動更新されます)  
必要な場合は、テキスト出力の設定をしてOSBと連動してください(デフォルトは改行で出力されます)

## 遭遇したトラブル/課題

* カメラが認識できない
```
Windowsのシステム設定-カメラのプライバシー設定
デスクトップアプリがカメラにアクセスできるようにするがONになっていて
FacialRecognitionSystem.exeの名前が出ているか確認してください
```
* カメラを選択したら場合エラーが出る場合 →　Visual C++ 再頒布可能パッケージのインストール（上部に記述）が必要
* マスク、サングラス着用などでは顔の認識率低下します。
* カメラとUSBキャプチャーのPC本体のUSBポートが一緒の場合は使用できないので、本システム使用のカメラは別途USBポートを使用してください
* OBSでプレーヤーの顔を配信しながら、本システムを使用することはできません（あくまでも顔認識だけの使用が求められます）


## 初期化

システムが立ち上がらなくなった場合は
"（インストールフォルダの）\\Data"　フォルダを(念のためバックアップ後)削除して、本システムを起動してください。
